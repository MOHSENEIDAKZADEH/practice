what is HTML ?

The HyperText Markup Language or HTML is the standard markup language for documents designed to be in a web browser. It is often assisted by technologies such as Cascading Sheets (CSS)and scripting languages such as JavaScript.


History

In 1980, physicist Tim Berners-Lee, a contractor at CERN, proposed and protoryped ENQUIRE, a system for CERN rescarchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system [3) Berners-Lee specified HTML and wrote the browser and server software in late 1990. That ycar, Berners-Lee and CERN data systems engineer Robert Cailliau collaborated on a joint request for funding, but the project was not formally adopted by CERN.
